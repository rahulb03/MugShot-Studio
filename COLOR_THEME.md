# MugShot Studio Color Theme

## Primary Colors

- **Primary Brand Color**: #0f7d70 (Teal Green)
- **Primary Background**: #FFFFFF (White)
- **Primary Text**: #000000 (Black)
- **Secondary Text**: #666666 (Dark Gray)
- **Accent Color**: #545b3a (Olive Green) - Used for buttons and highlights
- **Dot Color**: #0f7d70 (Teal Green) - Primary brand color
- **Dot Background**: #FFFFFF (White)

## Dark Mode Colors (if needed in future)

- **Primary Background**: #121212 (Dark Gray)
- **Primary Text**: #FFFFFF (White)
- **Secondary Text**: #CCCCCC (Light Gray)
- **Accent Color**: #545b3a (Olive Green)
- **Dot Color**: #FFFFFF (White)
- **Dot Background**: #121212 (Dark Gray)

## UI Component Colors

- **Button Background**: #545b3a (Olive Green)
- **Button Hover**: #43492f (Darker Olive Green)
- **Button Text**: #FFFFFF (White)
- **Input Background**: rgba(0, 0, 0, 0.05) (Light Gray with transparency)
- **Input Border**: rgba(0, 0, 0, 0.1) (Light Gray with transparency)
- **Dropdown Background**: #FFFFFF (White)
- **Dropdown Border**: rgba(0, 0, 0, 0.1) (Light Gray with transparency)
- **Dropdown Item Hover**: rgba(0, 0, 0, 0.05) (Very light gray)

## Status Colors

- **Success**: #4CAF50 (Green)
- **Warning**: #FFC107 (Amber)
- **Error**: #F44336 (Red)
- **Info**: #2196F3 (Blue)
